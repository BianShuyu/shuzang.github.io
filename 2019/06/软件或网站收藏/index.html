软件或网站收藏

shuzang  included in 我所热爱的生活啊 
2019-06-01  about 1173 words  3 min  pageviews  

Contents

Contents

win10：

  • snipaste：一般都用QQ截图，这一工具可以无需登录QQ，按F1键即可截图，该有的功能都有
  • ShareX：也是截图软件，可能比snipaste强一点，添加截图后OCR或者拼接等基本处理能力，添加自动上传图床的能力等
  • quicklook：单击空格预览文件内容
  • 致美化：win10美化社区
  • rainmeter：win10美化工具，基于效率考虑暂时只使用桌面时间插件
  • win10萌化通用工具箱：萌研社坑晨开发的win10美化工具，更改文件夹背景，使文件夹透明
  • 火绒安全软件：简洁而有用的电脑安全软件，彻底抛弃全家桶
  • Office Tool：帮助安装激活和更新office系列软件，以及提供干净的卸载功能，非常好用
  • PanDownload：百度网盘第三方工具，解除限速，还能直接搜资源，不过下载过多还是会被封。
  • IObit Uninstaller：软件管理工具，帮助删除卸载软件的残留文件
  • daemon tools lite：光盘刻录软件，界面简洁优雅，功能强劲
  • TranslucentTB：win10工具栏透明软件
  • 百分浏览器：本地化的Chrome，做了大量适合中国人改进的操作
  • Caesium：批量图片压缩
  • DiskGenius：磁盘划分
  • folder-explorer：分析文件目录，统计数据并以树形结构和图表的形式展示结果，也可以导出多种格式留存

Android：

  • 存储空间清理：用于清理手机空文件夹

chrome插件：

  • Google学术搜索按钮：查询文献，获取文献引用
  • Adblock Plus：广告拦截
  • HitUP-Find Top Things：在浏览器新标签页展示github流行项目
  • 划词翻译

网页工具：

  • CleverPDF：PDF和其它文件格式的转换，PDF删除页面，加解密，提取图像，压缩，拆分等功能
  • GPSspg：在线经纬度查询，在地图上点击地点或搜索栏搜索，可以查看指定地点的经纬度
  • Chinese Etymology：汉字字源搜索，能找到每个汉字演变的各个阶段，感受汉字的魅力。
  • 异步社区：计算机经典书籍推荐，网站简洁优雅，不一定在这里买书，但寻找某个领域的计算机经典书籍可以在这里找
  • docsify：文档站点生成，为产品写文档时可以挂靠在这里，写普通的技术文章也可以，没有限制。
  • 倾城之链：分享全球优秀网站，可以发现、学习很多有用或有趣的事物。
  • 路过图床：免费公共图床，所见过的最好用的，只是已经用了github做图床，所以不用它了。
  • beautiful.ai：更方便的创建PPT

网站

  • 行动村：公益项目组队，适合参加公益活动，但地点一般在上海、杭州
  • Microsculpture：一个很酷的网站，有各种昆虫的照片，每一只都可以放大看细节，非常精致和有趣，帮助我们了解昆虫的世界。
  • Mofan Python：从Python基础知识到数据处理，再到机器学习的一系列教程，写的通俗易懂。
  • C
    • 可视化C代码
    • C编程标准
  • Python
    • Python123
    • 慕课：Python语言程序设计
    • Python文档
  • 80x86汇编小站
  • IT学习路线
